# `superhuman setup`

Initialize `~/.superhuman/superhuman.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- CLI onboarding: [Onboarding (CLI)](/start/wizard)

## Examples

```bash
superhuman setup
superhuman setup --workspace ~/.superhuman/workspace
```

To run onboarding via setup:

```bash
superhuman setup --wizard
```
